# 🚀 ORBIT – AI Powered Space Tourism Platform

ORBIT is a futuristic full-stack space tourism web application designed to deliver a cinematic interplanetary travel experience. Users can explore luxury orbital destinations, reserve futuristic journeys, interact with an AI-powered concierge, and experience immersive sci-fi storytelling.

---

# 🌌 Features

- ✨ Cinematic futuristic UI
- 🌍 Interactive space destination explorer
- 🤖 AI Space Concierge powered by FastAPI
- 🛒 Dynamic reservation system using localStorage
- ⏳ Live mission countdown timer
- 🛰 Mission schedules & spacecraft details
- 🎥 Orbital experience gallery
- ☁️ Google Cloud Run deployment
- 📱 Fully responsive design
- 🌠 Smooth animations & glassmorphism effects

# ⚡ API Endpoints

## GET /destinations

Returns available luxury space destinations.

## GET /missions

Returns upcoming launch schedules and spacecraft availability.

## POST /reserve-trip

Processes orbital travel reservations.

## POST /chat

AI-powered futuristic travel concierge chatbot.

## POST /contact-submit

Handles user inquiries and reservations.

# ▶️ Run Locally

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/orbit-space-tourism.git
cd orbit-space-tourism
```

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

## 3️⃣ Run FastAPI Server

```bash
uvicorn main:app --reload
```

## 4️⃣ Open Browser

```bash
http://127.0.0.1:8000
```

---

# 🐳 Docker Run

## Build Docker Image

```bash
docker build -t orbit-app .
```

## Run Container

```bash
docker run -p 8080:8080 orbit-app
```
